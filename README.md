# OOP-bangun-ruang
Cetak luas permukaan dan volume bangun ruang di console

How to:

1. Buat instance.
2. Misal: kubus -> kubus = new BangunRuang()
3. Biarkan undefined
4. Cetak luas permukaan dan volume di console -> console.log(kubus.hitungKubus(2))
5. Lihat hasil keluaran di console
